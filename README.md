# 📦 Meli-Action - ملی اکشن

**ابزار ضدفیلترینگ قدرتمند با GitHub Actions**

با توجه به محدودیت‌ها و فیلترینگ شدید اینترنت در ایران، **ملی اکشن** به شما اجازه می‌دهد فایل‌ها، ویدیوها و صفحات وب را از طریق سرورهای GitHub (که فیلتر نیستند) دانلود کنید و مستقیماً داخل ریپازیتوری خودتان ذخیره کنید.

---

## ✨ قابلیت‌های اصلی

### 🎥 دانلود ویدیو
- **YouTube**: دانلود ویدیو در کیفیت دلخواه (حتی بالای ۲ گیگابایت)
- **Pornhub** و سایت‌های مشابه: دانلود مستقیم از لینک صفحه
- **Telegram**: دانلود فایل از کانال‌های عمومی

### 📥 دانلود فایل
- **لینک مستقیم** (هر نوع فایلی)
- **Google Play** (فایل‌های نصبی اپلیکیشن)
- **GitHub Releases** (دانلود از ریلیزهای گیتهاب)

### 🎵 موسیقی و صدا
- **Spotify**
- **SoundCloud**

### 🌐 آرشیو وب
- ذخیره صفحات وب فیلترشده به صورت **MHTML** کامل (با pyppeteer)

### 🚀 مدیریت فایل‌های حجیم (جدید)
- تقسیم خودکار فایل‌های بزرگ به پارت‌های ۹۵ مگابایتی
- پشتیبانی از آرشیو **7z** (تک‌پارت یا چندپارت، اختیاری رمزدار)
- آپلود هوشمند به **GitHub Repository** یا **GitHub Releases**

---

## 🚀 نحوه شروع (فقط ۲ دقیقه)

### ۱. فورک کردن ریپازیتوری
دکمه **Fork** را در بالای صفحه بزنید.

### ۲. فعال کردن مجوزهای Actions (الزامی)
1. به **Settings** ⚙️ بروید.
2. از منوی چپ: **Actions** → **General**
3. در بخش **Actions permissions** گزینه **"Allow all actions and reusable workflows"** را انتخاب کنید.
4. **Save** را بزنید.

> 💡 این تنظیم فقط روی ریپازیتوری فورک‌شده شما اعمال می‌شود و کاملاً امن است.

### ۳. استفاده از ابزارها
1. به تب **Actions** بروید.
2. workflow مورد نظر را از لیست سمت چپ انتخاب کنید (مثلاً `Download YouTube media`).
3. روی **Run workflow** کلیک کنید.
4. لینک و تنظیمات دلخواه را وارد کنید.
5. دوباره **Run workflow** را بزنید.

فایل‌های دانلودشده بلافاصله در ریپازیتوری شما (یا در Releases) ظاهر می‌شوند.

## 🛠 نکات مهم

- **فایل‌های خیلی بزرگ**: به صورت خودکار تقسیم و در صورت نیاز به صورت 7z فشرده می‌شوند.
- **یوتیوب**: برای جستجوی بهتر، فایل `google_service.json` را در تنظیمات v2ray خود استفاده کنید (فقط جستجو و تامبنیل کار می‌کند).
- **به‌روزرسانی**: برای دریافت آخرین تغییرات، در ریپازیتوری فورک‌شده روی **Sync fork** کلیک کنید.
